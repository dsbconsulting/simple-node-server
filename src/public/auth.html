Aspera Connect Simple Authentication Example

  

    
  

    
This example shows how to authenticate an user against a remote host with Aspera Connect 2.8+, using the API method authenticate.
For simplicity and clarity, this sample code does not check that all the fields are filled, and does not mask the password.
